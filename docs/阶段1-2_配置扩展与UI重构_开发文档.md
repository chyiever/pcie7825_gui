## 阶段1：配置参数扩展

### 开发内容

1. **DisplayParams扩展**
   - 添加`rad_enable: bool = False`参数
   - 实现显示数据的弧度转换控制

2. **TimeSpaceParams新增**
   - 创建完整的Time-Space图配置数据类
   - 包含窗口帧数、FBG范围、降采样、色图等参数

3. **AllParams结构更新**
   - 集成新的`time_space`参数组
   - 保持向后兼容性

### 关键修改

**文件**: `src/config.py`

```python
@dataclass
class DisplayParams:
    # ... 原有参数 ...
    rad_enable: bool = False  # 新增rad转换控制

@dataclass
class TimeSpaceParams:
    window_frames: int = 5
    distance_range_start: int = 40      # 对应FBG起始索引
    distance_range_end: int = 100       # 对应FBG结束索引
    time_downsample: int = 50
    space_downsample: int = 2
    colormap_type: str = "jet"
    vmin: float = -0.02
    vmax: float = 0.02

@dataclass
class AllParams:
    # ... 原有参数组 ...
    time_space: TimeSpaceParams = field(default_factory=TimeSpaceParams)  # 新增
```

### 技术要点

- **数据结构设计**: 采用dataclass确保类型安全和默认值管理
- **参数映射**: distance_range概念适配为FBG Idx range
- **兼容性保持**: 新参数添加不影响现有功能
